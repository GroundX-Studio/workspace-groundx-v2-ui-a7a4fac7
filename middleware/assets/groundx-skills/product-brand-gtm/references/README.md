# Product Brand GTM Reference Index

Use this index when the public GroundX Agent Harness is installed and the work
needs customer-facing GroundX product positioning, value propositions, proof points,
buyer framing, objections, or product descriptions.

## Fast Path

1. Start in `../SKILL.md` for the public routing contract.
2. Pick one reference from the map below.
3. Open `proof-points.md` before using any customer, benchmark, accuracy, or outcome
   claim.
4. Open `brand-relationship.md` whenever GroundX, EyeLevel, and Valantor naming or
   hierarchy comes up.
